## üîå Socket.IO Events

### Server-Side Event Handlers (4 total)

1. **`join-room`**
   - Input: `roomId` (string)
   - Action: User joins room, receives current state
   - Output: Emits `sync-state` with room data

2. **`video-action`**
   - Input: `{ roomId, type, value }`
   - Types: 'play', 'pause', 'seek'
   - Output: Broadcasts to all users in room

3. **`add-to-queue`**
   - Input: `{ roomId, video: { id, title } }`
   - Action: Adds video to room queue
   - Output: Broadcasts updated queue to all users

4. **`send-message`**
   - Input: `{ roomId, message, user }`
   - Action: Relays message to all users in room
   - Output: Emits with timestamp (Date.now())
